1.masuk ke aplikasi github desktop
2.pencet 'file' pada pojok kiri atas
3.pilih 'add local repository' 
4.pencet 'choose(pilih satu folder yang ingin dimasukkan)'
5.pencet 'add repository'
6.pencet link biru 'create a repository'
7.kemudian beri nama , jika sudah pencet 'create repository'
8.pencet 'publish repository' beri nama lalu pencet 'publish repository'
9.dan yap , jadilah repository tersebut , selamat!
